Network Analysis Simulation for Botnet Detection

This simulation demonstrates how to use statistical and visualization tools to analyze network traffic, comparing a normal scenario with a botnet attack scenario. Tools used include descriptive statistics, statistical tests, regression models, and time series analysis.

1. Descriptive Statistics of the Network

Theoretical Approach: Descriptive statistics help summarize network data, identifying patterns and anomalies. During a botnet attack, malicious nodes show an abnormal number of connections and a significant increase in traffic.

2. Kolmogorov-Smirnov Test

Theoretical Approach: This test compares the empirical distribution of network data with a theoretical distribution, such as the normal distribution. During an attack, the response time distribution tends to deviate from normality.

3. Regression Models

Theoretical Approach: Linear regression predicts traffic volume ("y") based on the number of connections ("x"). During an attack, the slope ("m") increases, indicating abnormal traffic growth.
